Properti CSS `border-right-width` digunakan untuk mengatur lebar border kanan suatu elemen.

## CSS Syntax
```css
border-right-width: medium | thin | thick | length | initial | inherit;
```

#### Javascript Syntax
```js
object.style.borderRightWidth="5px"
```
